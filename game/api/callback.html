支付回调

支付回调参数 使用post

CP 收到支付回调后返回 OK 将闭合订单

字段 类型 说明
result string SUCCESS 为成功 其它为失败
uid string 平台用户ID
transaction_id string 订单号
product_id int 商品ID
appId int 应用ID
isTest int 是否为测试 1为测试
extra string 自定义参数 64位以内
nonce_str string 随机字符串
sign string 签名
