签名

将key按照字母升序排列,然后使用&连接,再加上app私钥,最后计算md5值。如

appId 1
data 123
sign = md5(appId=1&data=123APP_SECREET)

token j832323ab
nonce 1234567890123456
sign = md5(token=j832323&nonce=1234567890123456APP_SECREET)
