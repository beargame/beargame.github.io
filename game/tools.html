设置工具

地址 http://test.g.ibeargame.com/index.php?c=open&a=tools

类型 描述
UserInfo 参数中 填写accessToken 可获取userInfo
SetGameUrl 填好 appId，appSecret ,参数中填写游戏地址
SetNotifyUrl 填好 appId，appSecret ,参数中填写游戏回调地址
TextToJson 支持将文本转换为json，格式见附1
SetProduct 填好 appId，appSecret ,参数中填写商品json

附1 每行为一个商品，商品 顺序为id price name desc 如果没有desc 也可以 id price name 转好的json 可以作为SetProduct的参数
price 单位 为分

1    2000    20000彩钻    20000彩钻
2    1000    10000彩钻    10000彩钻
3    500    5000彩钻    5000彩钻
